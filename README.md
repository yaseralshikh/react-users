# React + Prisma + Inngest – User Management Monorepo

This repository contains a small educational CRUD project for user management.

- `client/` – React + TypeScript front-end (Vite)
- `server/` – Node.js + TypeScript + Express + Prisma + Inngest back-end

## Getting Started (high level)

- `cd client` – front-end code and scripts
- `cd server` – back-end code and scripts

Detailed setup and commands will be added step-by-step.
